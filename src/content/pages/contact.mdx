# Contact

Vous avez des questions sur HexaGlue ? Plusieurs moyens de nous joindre :

## GitHub

Le meilleur endroit pour :

- **Signaler un bug** : [Ouvrir une issue](https://github.com/hexaglue/hexaglue/issues/new)
- **Proposer une fonctionnalité** : [Discussions](https://github.com/hexaglue/hexaglue/discussions)
- **Contribuer au code** : [Guide de contribution](https://github.com/hexaglue/hexaglue/blob/main/CONTRIBUTING.md)

## Entreprise

Pour les demandes professionnelles :

- **Accompagnement** sur la mise en place de l'architecture hexagonale
- **Formation** sur le DDD et la Clean Architecture
- **Support** pour l'intégration de HexaGlue

Contactez-nous par email : [info@hexaglue.io](mailto:info@hexaglue.io)
